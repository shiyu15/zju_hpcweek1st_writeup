# ZJU HPC1 高性能优化比赛题目和题解

个人第一届浙江大学高性能计算优化比赛（ZJU HPCweek）的题目和题解的仓库。

这次比赛很有意思。以并行为背景出了很多cpu上面的优化题目。我运气不错，拿到了总榜第一。也是第一次在类似的赛事中拿到这么高的排名。这是我的代码和题解，欢迎大家交流。

仓库中每个文件夹对应一个优化任务，并附有相应的题解文档（solve.md）。
